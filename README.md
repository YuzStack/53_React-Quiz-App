# React Quiz App 🧠

![React Quiz App Homepage](/react-quiz-app.png)

The React Quiz App is an interactive platform built to test knowledge of React.js through a clean, structured interface. The project focuses on state management and providing a cohesive user experience from the start of the quiz to the final results.

**[Live Preview](https://yuzstack-react-quiz-app.netlify.app/)**

## 🛠️ Technical Stack

- **Frontend Library:** React.js
- **Styling:** Tailwind CSS

## 📝 Development Overview

This project was an independent build focused on architecting a reliable quiz engine. I prioritized the flow of the application, ensuring that user progress is tracked accurately and that the final score calculation is precise.

## ✨ Key Features

- **Dynamic Quiz Flow:** Manages a variety of questions and multiple-choice options, transitioning smoothly between stages.
- **Progress Tracking:** Includes a visual progress bar that updates as the user moves through the quiz, providing a clear sense of completion.
- **Final Score Calculation:** Aggregates user performance throughout the session and displays the total score only upon completion of the quiz.
- **Modern UI with Tailwind:** Leverages Tailwind CSS for a highly responsive, utility-first design, ensuring the app is accessible on all device sizes.
- **Modular Architecture:** Built using reusable React components to ensure the codebase remains clean, maintainable, and scalable.

## 🚀 Getting Started

To run this project locally:

1. Clone the repository: `git clone https://github.com/YuzStack/53_React-Quiz-App.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
